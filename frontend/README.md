# Frontend

Frontend for realtime whiteboard sharing app.

## Features

* Interactive canvas
* Real-time updates
* Chat sidebar
* Responsive design

## Run

```bash
npm install
npm run dev
```

## Folder Structure

```txt
Frontend/
├── components/
├── pages/
├── types/
└── App.tsx
```

## Tools

* React
* TypeScript
* Tailwind CSS
* Socket.IO Client
